Fix: Task Manager Not Opening

Problem: Clicking Task Manager shows nothing or crashes.

Fix 1: Press Ctrl + Shift + Esc directly.

Fix 2: Run sfc /scannow in CMD (Admin).

Fix: WiFi Connected but No Internet

Problem: WiFi shows connected but websites don't load.

Fix 1: Run netsh winsock reset in CMD (Admin).

Fix 2: Set DNS to 8.8.8.8 and 8.8.4.4.

Fix: File Explorer Not Responding

Problem: Explorer.exe hangs or crashes often.

Fix 1: End all "Windows Explorer" tasks and restart it.

Fix 2: Disable Quick Access recent files.

Fix: No Sound After Update

Problem: System has no audio after Windows Update.

Fix 1: Set the correct output device in sound settings.

Fix 2: Roll back or update the audio driver.

Fix: System Restore Fails

Fix: Laptop Not Shutting Down

Problem: Clicking Shut Down just restarts or hangs.

Fix 1: Disable Fast Startup in Power Options.

Fix 2: Use shutdown /s /f /t 0 via CMD.

Fix: High CPU Usage (System Idle)

Problem: Task Manager shows high CPU but system is idle.

Fix 1: Ignore "System Idle Process" – it’s normal.
